# Campaign Dashboard - Frontend

This is the Vue 3 frontend application for the Campaign Dashboard assessment.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Project Structure

```
src/
├── views/          # Page components
├── components/     # Reusable components (create these)
├── stores/         # Pinia stores (create these)
├── services/       # API services (create these)
├── router/         # Vue Router configuration
├── App.vue         # Root component
└── main.js         # Application entry point
```

## Development Notes

- Make sure the backend server is running on `http://localhost:3000`
- See `../ASSESSMENT.md` for detailed requirements and API endpoint information
